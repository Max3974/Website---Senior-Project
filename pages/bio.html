Personal Biography

This is a brief, formal biography that will appear in the JHC annual report. Please provide several succinct sentences outlining education, experience, focus of research, notable awards and achievements, etc.

This does not need to be edited every reporting period. If no changes are made, the Progress Report System will automatically use the previous version of your bio.

If you wish, this updated bio may also be used as your public bio on the CCOM Website. If you would like this to be used, or would like to make changes to the website, please contact Colleen Mitchell at colleen@ccom.unh.edu
