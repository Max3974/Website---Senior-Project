Create Seminars and Presentations

This section is for seminars and presentations you have GIVEN, not attended. Enter any named talk or presentation you have given at conferences, workshops, visitations, or seminars. This talk could have been given at JHC/CCOM or abroad. Do NOT enter course lectures or other instructional presentations here. Also: remember that if you had an abstract for this presentation or talk, please enter it as a "Conference Abstract" in the Publications module on the CCOM/JHC website.

Section notes:

  • Title: name of seminar or talk
  • NOAA Related: Check if NOAA was involved in any way (other than NOAA OCS being the Center's primary funding agent). This includes any NOAA personnel working with you, visiting you, or doing this for or with any NOAA division or group.
  • Tagged CCOM Members: enter any personnel here only if they gave the presentation with you (coordinators don't count). As you type, a drop-down menu will automatically populate (when you stop typing). If the person’s name appears in the menu, click on it to add them. You may do this for multiple individuals. Do not worry about trying to tag people outside CCOM or JHC. This is not a required field.
  • Type of Presentation: select an option as to what type of presentation it was.
  • Organization: Organization you gave the talk to. This section will auto-populate; utilize existing entries if you can, otherwise type in the info and move to the next field.
  • Event: If this talk was at an event, enter the name of the event here. This section will auto-populate so utilize existing entries if you can, otherwise type in the info and move to the next field.
  • Date Given: Enter the date of the presentation. If it lasted more than one day, check the box labeled "Show End Date" and a second Date field will appear.
  • Location: Enter the city, state (if applicable) and country where the presentation was held.
  • Description: Write a general description here in two or three sentences. Keep this section brief — if there are specifics, or broader topics to discuss where this event is concerned, please put that info in the Narrative section of your report.
